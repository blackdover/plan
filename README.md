# plan
一个简单的做计划的应用  
## 使用方法：  
**在页面右边的release下载**
快捷键  
Alt+a 快速添加   
Alt+d 快速删除（需要鼠标先选中）  
Alt+e 快速编辑（需要鼠标先选中）  
**更快速的编辑**：鼠标直接在想修改的任务上点两下，修改完可以回车，也可以双击最后一个任务的下一行（空白隐藏行）  
**移动任务**：鼠标按住，然后拖动指定任务，会移动到鼠标末位置的任务的上方
